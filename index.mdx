Gemma 3 Inference Engine is a minimal, readable PyTorch implementation of LLM inference modeled after vLLM's core ideas. It runs the Gemma-3-270M model with a block-based paged KV cache, a continuous batching scheduler, and an OpenAI-compatible REST API — all in a codebase designed to be read and extended.

## Get started in three steps

<Steps>
  <Step title="Install dependencies">
    ```bash
    python -m pip install torch tokenizers safetensors huggingface_hub fastapi uvicorn pydantic starlette
    ```
  </Step>
  <Step title="Start the API server">
    ```bash
    python -m openai_api.run
    ```
    The server loads the Gemma-3-270M weights on startup and exposes `POST /v1/chat/completions` on port 8000.
  </Step>
  <Step title="Send a chat request">
    ```bash
    python query_fastapi.py --stream --prompt "Give me one short line about LLM inference."
    ```
    Or use any OpenAI-compatible client pointed at `http://localhost:8000`.
  </Step>
</Steps>

<Note>
  You need approved access to the Gemma checkpoints on Hugging Face before the first download. See [Quickstart](/quickstart) for details.
</Note>
