---
title: "Get started with Gemma 3 Inference Engine"
description: "Install dependencies, get Hugging Face model access, and run your first Gemma-3 inference in five steps using direct generation or the API server."
---

This guide walks you from a fresh environment to a running inference session. You can either run generation directly with `python main.py` or serve the OpenAI-compatible API with `python -m openai_api.run`. Both paths are covered below.

<Note>
  You need approved access to the Gemma checkpoints on Hugging Face before the first download. Visit [google/gemma-3-270m-it](https://huggingface.co/google/gemma-3-270m-it) and request access. Once approved, run `huggingface-cli login` with your token.
</Note>

<Steps>
  <Step title="Install Python dependencies">
    Install the required packages with pip:

    ```bash
    python -m pip install torch tokenizers safetensors huggingface_hub fastapi uvicorn pydantic starlette
    ```

    For the optional benchmark scripts, also install:

    ```bash
    python -m pip install pandas
    ```

    <Tip>
      Python 3.10 or later is required. The engine uses built-in type syntax like `list[int]` and `set[int]`.
    </Tip>
  </Step>

  <Step title="Get Hugging Face model access">
    The engine downloads weights from `google/gemma-3-270m-it` (instruct) or `google/gemma-3-270m` (base) on first run.

    1. Go to [google/gemma-3-270m-it](https://huggingface.co/google/gemma-3-270m-it) and request access
    2. Once approved, authenticate your environment:

    ```bash
    huggingface-cli login
    ```

    If you already have the model files locally, place them in a directory named `gemma-3-270m-it/` (for the instruct model) or `gemma-3-270m/` (for the base model) in the project root. The engine will use local files if `model.safetensors` and `tokenizer.json` are present, skipping the download.
  </Step>

  <Step title="Run direct inference">
    The simplest way to test the engine is with `python main.py`. This loads the model, tokenizes a prompt, and streams output tokens to stdout:

    ```bash
    python main.py
    ```

    The default `RunConfig` uses these settings:

    ```python
    @dataclass
    class RunConfig:
        choose_model: str = "270m"
        use_instruct_model: bool = True
        prompt: str = "Give me a short introduction to large language models."
        max_new_tokens: int = 180
        temperature: float = 0.8
        top_p: float = 0.9
        top_k: int = 50
        repetition_penalty: float = 1.1
    ```

    Edit `RunConfig` in `main.py` to change the prompt or sampling parameters.

    <Tip>
      Device selection is automatic: CUDA is used if available, then Apple MPS, then CPU. GPU memory usage is printed after generation when running on CUDA.
    </Tip>
  </Step>

  <Step title="Start the API server">
    Start the OpenAI-compatible HTTP server:

    ```bash
    python -m openai_api.run
    ```

    The server loads the model during startup (not on the first request). Wait for the model to finish loading before sending requests — you can poll the readiness probe:

    ```bash
    curl http://localhost:8000/readyz
    # Returns {"status": "ready"} when the model is loaded
    ```
  </Step>

  <Step title="Query the API">
    Use the included query script:

    ```bash
    python query_fastapi.py --stream --prompt "Give me one short line about LLM inference."
    ```

    Or use `curl` directly:

    ```bash
    curl -X POST http://localhost:8000/v1/chat/completions \
      -H "Content-Type: application/json" \
      -d '{
        "model": "gemma-3-270m-it",
        "messages": [{"role": "user", "content": "What is LLM inference?"}],
        "max_tokens": 100
      }'
    ```

    You can also use any OpenAI-compatible client by pointing it at `http://localhost:8000/v1`:

    ```python
    from openai import OpenAI

    client = OpenAI(base_url="http://localhost:8000/v1", api_key="unused")
    response = client.chat.completions.create(
        model="gemma-3-270m-it",
        messages=[{"role": "user", "content": "What is LLM inference?"}],
        max_tokens=100,
    )
    print(response.choices[0].message.content)
    ```
  </Step>
</Steps>
